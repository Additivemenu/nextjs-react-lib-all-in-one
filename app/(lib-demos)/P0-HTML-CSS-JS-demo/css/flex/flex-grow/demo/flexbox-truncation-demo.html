/* The solution: */ .text-wrapper { flex: 1; /* Take available space */ min-width: 0; /* ‚ú® The magic property! */ } .text { white-space: nowrap; overflow: hidden; text-overflow: ellipsis; /* Now it works! */ }

üîë Key Takeaway

When using text-overflow: ellipsis inside a flex container, you must set min-width: 0 on the flex item containing the text. Otherwise, the flex item won't shrink below its content size, and the text will overflow instead of truncating.

This is one of the most common flexbox gotchas and applies to both min-width: 0 (horizontal) and min-height: 0 (vertical) scenarios!
